== Description ==

Protect your website by blocking unwanted traffic from Tor Exit Nodes. This plugin helps enhance your site's security by preventing access from these anonymous network endpoints, ensuring better control over who can interact with your content.

Upload the Torminator plugin to your website and activate it. The plugin will do the rest.
